The novation does not make the ACM a principal. ASC 606-10-55-39 tests control before transfer. The ACM never controlled the repo at any point — not pre-trade, not post-novation. The "fully liable" language in Rule 8 is a clearing rule requirement, not an accounting reclassification trigger.

The daily P&L from collateral marks belongs entirely to the EFC. The ACM is a conduit for the Funds-Only Settlement flow. This is the clearest behavioral test of agent classification under ASC 606: if you bear the upside/downside of the mark, you are principal. The ACM does not.

EFC default scenarioACM must fund FICC ($100.0153). ACM then holds a claim against EFC under Rule 8 §10. This triggers credit loss recognition under ASC 326 — but only if EFC defaults.
If EFC defaults: The ACM's contingent obligation to FICC crystallizes into an actual liability. The ACM funds FICC, then pursues the EFC. The loss provision at that point is an ASC 326 credit loss on the EFC receivable — not a repo mark-to-market loss, because the ACM never held the repo asset.

The one-line answer: The only item that belongs on the ACM's balance sheet is the fee receivable (~$0.001 per $100 repo). Everything else — the $100 repo asset, the $100 repo liability, the collateral marks, the NIM — belongs to the EFC or passes through without recognition. The FICC-facing obligation after novation is real but contingent: it becomes a balance sheet item only if the EFC defaults, at which point it is a credit loss under ASC 326, not a repo position.
